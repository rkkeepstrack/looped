Looped

Audio looper for macOS — waveform, seamless A/B loops, speed & pitch control.

Download for macOS

Or install with Homebrew

brew tap rkkeepstrack/looped https://github.com/rkkeepstrack/looped.git
brew install --cask --no-quarantine looped

Requires macOS 15+. Looped isn’t notarized, so a plain download needs xattr -dr com.apple.quarantine on the app (Homebrew’s --no-quarantine does this for you). Source on GitHub.
